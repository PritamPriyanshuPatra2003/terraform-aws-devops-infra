# Terraform AWS DevOps Infrastructure

## Project Overview

This project demonstrates how to provision AWS infrastructure using **Terraform**, following Infrastructure as Code (IaC) principles.

Instead of manually creating AWS resources through the AWS Management Console, the entire infrastructure is defined in Terraform configuration files, making deployments automated, repeatable, and version-controlled.

---

## Project Objectives

- Provision AWS infrastructure using Terraform
- Learn Infrastructure as Code (IaC)
- Automate infrastructure deployment
- Follow DevOps best practices
- Build a production-style Terraform project
